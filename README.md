# 使用方法

## CCW

1. 前往扩展页面(具体页面请见每个扩展下的use.md)
2. 点击**安装到我的工程**
3. 点击**复制ID**
4. 回到你要使用扩展的项目
5. 点击**扩展库**按钮
6. 点击**自定义扩展**
7. 点击加载自定义扩展的**安装**按钮
8. 在弹出的对话框中复制刚刚的ID
9. 完成
